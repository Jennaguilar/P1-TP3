TRABAJO PRACTICO N° 2 – PROGRAMACION I
INFORME

Incorpore el uso de Pilas y Colas a nuestro sistema educativo, utilizando Listas Enlazadas como estructura base.
Además, se agregaron nuevas funcionalidades, entidades y métodos, junto con métodos recursivos aplicados a estas estructuras.

→Implementaciones Realizadas
Pila (LIFO – Last In First Out)
Implementada usando ListaEnlazada.
Operaciones principales:
push(), pop(), peek(), isEmpty(), size().

Se usa en:
Historial de Cambios de Notas
Tareas Pendientes del Estudiante

✔️ Cola (FIFO – First In First Out)
Implementada usando ListaEnlazada.
Operaciones principales:
insertar(), quitar(), verPrimero(), estaVacia(), size().

Se usa en:
Gestor de Inscripciones
Turnos de Consulta con Profesores




Nuevas Entidades

Funcionalidades Implementadas
HistorialCambios (Pila)
Permite registrar cambios con registrarCambio().
Permite deshacer cambios con deshacerUltimoCambio().
 Es útil porque el último cambio es el primero en revertirse (LIFO).

GestorInscripciones (Cola)
Los estudiantes se atienden en orden de llegada.

Métodos:
agregarSolicitud()
verSiguienteSolicitud()
procesarSiguienteSolicitud()
Perfecto para un sistema donde se respeta el turno.

TareasPendientes en Estudiante (Pila)
agregarTarea()
verProximaTarea()
completarUltimaTarea()

Turnos de Consulta en Profesor (Cola)
agregarEstudianteACola()
verSiguienteEnCola()
atenderSiguienteEstudiante()



Métodos Recursivos Implementados
Ejemplos incluidos:
contarElementosPilaRecursivo()
vaciarColaRecursivo()
buscarEnPilaRecursivo()
(implementados dentro de Pila/Cola o usando funciones auxiliares)


Ejemplos de Uso
Todos los ejemplos están incluidos en App.java:
Registrar y deshacer cambios
Agregar y completar tareas
Agregar turnos y atenderlos
Agregar solicitudes y procesarlas



Conclusión
El TP integra Pilas, Colas, Recursividad y Listas Enlazadas dentro de un mismo sistema educativo.
Cada funcionalidad fue implementada usando la estructura adecuada:
Pila: acciones recientes o tareas
Cola: procesos secuenciales en orden de llegada
